Projects

Song Scoop

Song Scoop is a playlist app that allows anyone at your event to join a playlist session, add songs, and vote on the song queue order. It was made with parties, roadtrips, and video streaming in mind.

Song Scoop uses Vue with a Node backend, and utilizes the Spotify Web API and Web Player SDK through multiple authorization requests.

You can try the app (looks best in a mobile view), or see the code on Github.
